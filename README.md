## üöÄ Installation

### Chrome Web Store
1. Visit [BibliaTab on Chrome Web Store](#) _(link will be added after approval)_
2. Click **"Add to Chrome"**
3. Click **"Add Extension"** to confirm
4. Open a new tab to start using BibliaTab!

### Firefox Add-ons
1. Visit [BibliaTab on Firefox Add-ons](#) _(link will be added after approval)_
2. Click **"Add to Firefox"**
3. Click **"Add"** to confirm
4. Open a new tab to start using BibliaTab!

### Manual Installation (Developer Mode)

#### Chrome
1. Download the latest release from [GitHub Releases](https://github.com/guruskill7/bibliatab/releases)
2. Extract the ZIP file
3. Open Chrome and go to `chrome://extensions/`
4. Enable **"Developer mode"** (top-right toggle)
5. Click **"Load unpacked"**
6. Select the extracted folder
7. BibliaTab is now installed!

#### Firefox
1. Download the latest release from [GitHub Releases](https://github.com/guruskill7/bibliatab/releases)
2. Extract the ZIP file
3. Open Firefox and go to `about:debugging#/runtime/this-firefox`
4. Click **"Load Temporary Add-on"**
5. Select the `manifest.json` file from the extracted folder
6. BibliaTab is now installed!

## üõ†Ô∏è Technical Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **APIs**: 
  - Bible API (bolls.life) for verse data
  - Open-Meteo for weather information
  - Picsum Photos for background images

- **Storage**: Local Browser Storage API (no server, no tracking)
- **Cross-browser**: Automatic API detection for Chrome/Firefox compatibility
